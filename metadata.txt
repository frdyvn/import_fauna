about=Importe les fichiers CSV tabulés exportés depuis le portail Fauna (observatoire-fauna.fr) vers QGIS.

    Fonctionnalités :
    - Détection automatique des fichiers CSV Fauna dans un dossier (points, linéaires, polygones) par analyse du contenu, quel que soit le nom des fichiers
    - Prise en charge de plusieurs fichiers par type de géométrie (ex. point_site1.csv, point_site2.csv)
    - Affichage de la liste des fichiers détectés avec type de géométrie et nombre d'entités, sélection individuelle avant import
    - Jointure optionnelle des métadonnées (Metadonnees.csv) directement dans la table attributaire des couches (clé IdJdd ↔ IdJeuDonnees) ; plusieurs fichiers de métadonnées fusionnés automatiquement avec déduplication
    - Export optionnel en GeoPackage (.gpkg) : dans ce cas les couches sont chargées depuis le fichier plutôt que comme couches temporaires
    - Toutes les couches sont regroupées dans un groupe nommé d'après le dossier source et le zoom est automatiquement ajusté
    - Projection Lambert 93 (EPSG:2154)
